## Что нового в версии 1.1(2) 🚀

В этом обновлении мы исправили критические ошибки компиляции, оптимизировали процесс сборки и подготовили приложение к релизу в магазинах.

### Исправления и улучшения:
* **Фикс ошибок окружения Java:** Устранили проблемы со сборкой, завязав её на автоматическую конфигурацию, независимую от внешних репозиториев OpenJDK.
* **Переход на релизный режим:** Исправили ошибку с флагом отладки — теперь `isDebuggable = false` выставлен по умолчанию, и приложение работает в полноценном продакшн-режиме.
* **Гибкая цифровая подпись:** Настроили автоматический откат на `debugConfig` при локальной компиляции, чтобы сборка больше не падала из-за отсутствия серверных ключей.
* **Оптимизация APK:** Убрали лишние дебаг-логи, уменьшив нагрузку на систему и повысив плавность работы дневника.
